# Examples Setup

## iOS

1. If you don't have bundler gem installed:

```
gem install bundler
```
If you don't have the repo cloned locally

2. ```
git clone repo
cd example/ios
```

3. Install dependencies and open the workspace:

```
npm install
npm run build:ios
npm run run:ios
```

or

```
yarn install
yarn build:ios
yarn run:ios
```

## android

1. Start your emulator

2. Install dependencies and run the example:

```
npm install
npm run run:android
```

or

```
yarn install
yarn run:android
```
